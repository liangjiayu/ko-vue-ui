# ko-vue-ui

一款`vue`组件库



## 特性

- 支持`ts`
- 轻量级



## 安装

```bash
npm i ko-vue-ui -S
```



## 快速上手

```js
import Vue from 'vue'
import koui from 'ko-vue-ui'
import 'ko-vue-ui/dist/ko-vue-ui.css';

Vue.use(koui)
```
